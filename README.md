# Lunar Archive

## 1. Về dự án
Lunar Archive Project là một dự án lưu trữ dữ liệu trên server được thành lập và quản lý của Phatcraft.<br>
Dự án được thiết kế để hoạt động trên Raspberry Pi Zero 2W.
